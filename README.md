# Happy Bot Kit CAD Files

Welcome to the Happy Bot Kit!

These CAD files are part of Team Happy Bots' mission to help families discover the amazing world of combat robotics. We started with zero experience, learned by building together, and created these resources to help others get started faster.

## About Happy Bot Kit

Happy Bot Kit is designed to provide families with a foundation for building, learning, and competing in combat robotics. The project focuses on making robotics approachable for beginners while teaching real engineering, design, and problem-solving skills.
